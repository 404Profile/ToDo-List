# ToDo List

Этот репозиторий для списка задач. Код полностью открыт. Чтобы приступить к настройке приложения на вашем компьютере, ознакомьтесь с приведенным ниже руководством по установке.

## Требования

Для начала установки требуются следующие инструменты и зависимости.

- .NET 8.0
- Минимальная версия Visual Studio и MSBuild: 17,7+

Больше информации для [требований к версии пакета SDK для .NET 8](https://learn.microsoft.com/ru-ru/dotnet/core/compatibility/sdk/8.0/version-requirements)

## Установка

> Обратите внимание, что вы можете настроить местоположение "../dir/ToDo-List" в любом каталоге на вашем компьютере, который вам нужен.

1. Клонируйте этот репозиторий с помощью `git clone https://github.com/404Profile/ToDo-List.git ../dir/ToDo-List`
